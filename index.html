The Pioneer's FAQ

Stuck on "Connecting"?

This is the most common hurdle! There are 3 main reasons:

  • Launch Order: Always start the Accio Desktop App BEFORE opening Chrome.
  • Developer Mode: Chrome MUST be in "Developer Mode" with the extension folder loaded manually.
  • Port Conflict: Some security software blocks 127.0.0.1:8000. Download our Fix Patch to solve this.
